# Project - Paytm Clone

## Features 
- User can Signup, Login, Update user Data.
- User can see other users, Can search them with filter.
- User can Trasfer Money to other user. 


## Tech Used 
- Typical MERN stack
- Node.js
- MongoDB
- React.js 
- Recoil 
- Express.js
- Tailwind
